# GamersOn API

GamersOn é uma API construída com o intuito de dispor um cadastro de games e avaliações.

Essa API foi construída para fins de estudo sobre: 

- .NET 6
- Clean Architecture 
- CQRS e padrão mediator
- Entity Framework Core
- Principios do Solid
- Fluent Validation (Próximas implementações)
- Pipeline Behavior (Próximas implementações)
- Testes unitários  (Próximas implementações)
